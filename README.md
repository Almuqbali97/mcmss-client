# Research Ethics Platform - Frontend

React frontend application built with Vite.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Project Structure

```
src/
├── components/
│   ├── Login.jsx           # Login page
│   ├── Dashboard.jsx        # User dashboard
│   ├── SubmissionForm.jsx  # Multi-step submission form
│   ├── ViewSubmission.jsx  # View submission details
│   └── AdminPanel.jsx     # Admin panel
├── utils/
│   └── api.js              # API utility functions
├── App.jsx                 # Main app component with routing
├── App.css                 # Global styles
└── main.jsx               # Entry point
```

## Development

The app runs on `http://localhost:5173` by default. Make sure the backend server is running on `http://localhost:3001`.
